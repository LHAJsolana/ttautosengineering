# Mercedes Electronics: Used-Buyer Notes

Mercedes-Benz ownership can be excellent when the car is electrically healthy.

But electronics should not be judged only by whether the dashboard is clean. Stored faults, low-voltage history, intermittent module behavior, and weak batteries can all affect the buying decision.

## What to check

Before purchase, check:

- Battery age and test result
- Charging system behavior
- Stored module faults
- Seat, window, mirror, and lock operation
- Infotainment function
- Warning messages after a warm test drive
- Water ingress signs

Electrical faults can be more expensive to diagnose than to repair.

## Practical rule

Scan the car fully. A clean dashboard is not the same as a clean diagnostic report.
